Título do Exercício: Criação de um site de portfólio pessoal

Descrição:

Desenvolva um site de portfólio pessoal utilizando HTML, CSS e JavaScript. O objetivo deste exercício é aplicar os conceitos básicos de desenvolvimento web para criar uma página que apresente informações sobre você, suas habilidades e projetos.

Requisitos:

1. Estrutura HTML:
   - Crie uma página com as seguintes seções:
     - Cabeçalho: Deve incluir um título e um menu de navegação com links para as seções da página.
     - Sobre Mim: Uma breve descrição pessoal, incluindo formação, interesses e objetivos.
     - Projetos: Apresente pelo menos dois projetos, com nome, descrição e, se possível, links ou imagens.
     - Contato: Um formulário que permita aos visitantes entrarem em contato com você.
   
2. Estilização CSS:
   - Utilize um arquivo CSS externo para estilizar a página.
   - Aplique estilos que melhorem a apresentação visual, como cores, fontes e layouts.
   - O design deve ser responsivo, adaptando-se a diferentes tamanhos de tela (desktop e mobile).

3. Funcionalidades JavaScript:
   - Implemente interatividade no site, como:
     - Validação dos campos do formulário de contato antes do envio.
     - Exibição de uma mensagem de confirmação após o envio do formulário.
     - Outros efeitos interativos a seu critério (por exemplo, animações, slideshows).

4. Código Comentado:
   - Adicione comentários no código HTML, CSS e JavaScript explicando o propósito das principais partes e trechos de código.

5. Organização dos Arquivos:
   - Organize os arquivos do projeto em pastas (por exemplo, `css`, `js`, `imagens`).
   - Nomeie os arquivos de forma clara e consistente.
