Sobre mim

Me chamo Heloisa Cabral, graduanda do curso de Engenharia de Software pela Universidade de Vassouras, desenvolvedora Front-end e Designer Gráfico.

Desenvolvo páginas, sistemas, peças gráficas responsívas e de layout atrativo, materiais publicitários, logos e identidades visuais.
Tenho uma trajetória na área de tecnologia de mais de 3 anos quando comecei, por conta própria, a estudar programação.

Dentro do mercado de trabalho, atuei como jovem aprendiz responsável pelo desenvolvimento de sistemas no TRAINSTATION UNIFAA e, logo depois, como Designer Gráfico no setor de Comercial e Marketing da mesma instituição.

Sou apaixonada pela ligação entre a computação e a arte que as IAS Generativas desempenham.
Como Designer, atuo principlamente no ramo de Produtos para Associações Atléticas Universitárias.

Estou no caminho para me formar como Engenheira de Software para começar a lecionar na área de Inteligencia Artificial.

Além da tecnologia, trabalho realizando traduções de artigos e textos acadêmicos do português para o inglês.

Conhecimentos

HTML5

HTML5 é uma linguagem de marcação de hipertexto, que usamos para marcar os elementos da página e construí-la.


CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.


JavaScript

JavaScript é uma linguagem de programação interpretada usada no desenvolvimento para descrever funcionalidades e comportamentos da página.

MidJourney e CHATGPT

Inteligência artificial generativa ou IA generativa é um tipo de sistema de IA capaz de gerar texto, imagens ou outras medias em resposta a solicitações em linguagem comum.


Adobe Photoshop

Adobe Photoshop é um software caracterizado como editor de imagens bidimensionais do tipo raster desenvolvido pela Adobe Systems. É considerado o líder no mercado dos editores de imagem profissionais, assim como o programa de facto para edição profissional de imagens digitais e trabalhos de pré-impressão.


Adobe Illustrator

O Adobe Illustrator é a ferramenta de design gráfico líder do setor que permite criar tudo o que você imaginar, de logotipos e ícones a imagens e ilustrações, e personalizar sua criação com precisão de nível profissional, e oferece recursos para poupar tempo, como repetição de padrões ou edições globais.
